Política de Privacidade

O FinanPro salva os dados financeiros no próprio navegador do usuário, usando armazenamento local. Isso significa que, nesta versão, os dados não são enviados automaticamente para um servidor.

O usuário é responsável por exportar seus dados quando desejar fazer backup. Se uma versão futura incluir conta, login ou backup em nuvem, esta política deverá ser atualizada.

O app não substitui consultoria financeira, contábil, jurídica ou de investimentos. As análises são informativas e devem ser avaliadas conforme a realidade de cada pessoa.
