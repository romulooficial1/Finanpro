Política de Privacidade

O FinanPro respeita a privacidade dos usuários e utiliza os dados informados apenas para oferecer recursos de controle financeiro, sincronização na nuvem e melhoria da experiência.

  • Dados financeiros cadastrados são usados para exibir dashboard, metas, orçamentos e relatórios.
  • O login pode ser feito por e-mail/senha ou conta Google.
  • Dados de assinatura são usados para liberar recursos Premium; dados de cartão e Pix devem ser processados pelo gateway de pagamento.
  • O site pode utilizar cookies e tecnologias semelhantes para funcionamento, segurança e análise.
  • O FinanPro não vende dados pessoais dos usuários.

Termos de Uso e Isenção

Ao utilizar o FinanPro, o usuário concorda em usar a plataforma de forma responsável e informar dados verdadeiros.

  • As informações exibidas têm caráter organizacional e informativo.
  • O FinanPro não substitui consultoria financeira, contábil, jurídica ou profissional.
  • O usuário é responsável pelas decisões tomadas com base nos dados cadastrados.
  • Recursos Premium dependem de assinatura ativa confirmada pelo gateway e verificada na conta do usuário.
  • Funcionalidades podem ser atualizadas para melhorar segurança e desempenho.

Sobre Nós

O FinanPro nasceu com o objetivo de ajudar pessoas a organizarem receitas, despesas, metas e orçamentos de forma simples, moderna e acessível.

O projeto foi criado por Rômulo para reunir em um único lugar controle financeiro, relatórios PDF, sincronização na nuvem e análises inteligentes.

FAQ

  • O FinanPro é gratuito? Sim. Há plano Gratuito e plano Premium por R$ 9,90/mês.
  • Meus dados ficam salvos? Sim, ao entrar na conta, os dados podem ser sincronizados na nuvem.
  • Posso acessar de outro celular? Sim, basta usar a mesma conta.
  • Posso gerar PDF? Sim. No Gratuito você gera 1 PDF básico por mês; no Premium, relatórios avançados ficam liberados.

Segurança dos Dados

A segurança das informações dos usuários é prioridade. O FinanPro utiliza autenticação e armazenamento em nuvem para manter os dados vinculados à conta do usuário.

  • O acesso é protegido por autenticação.
  • Os dados são separados por usuário.
  • O usuário deve manter sua senha e conta Google protegidas.
  • Informações não são vendidas para terceiros.
