Segurança dos Dados

A segurança das informações dos usuários é prioridade no FinanPro.

  • O acesso é protegido por autenticação.
  • Os dados são vinculados à conta do usuário.
  • As informações são separadas por usuário para preservar privacidade.
  • O usuário deve manter sua senha e conta Google protegidas.
  • O FinanPro não vende informações pessoais para terceiros.
