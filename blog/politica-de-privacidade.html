Política de Privacidade

O FinanPro respeita a privacidade dos usuários e utiliza os dados informados apenas para oferecer recursos de controle financeiro, sincronização na nuvem e melhoria da experiência.

  • Os dados financeiros cadastrados são usados para exibir dashboard, metas, orçamentos, análises e relatórios.
  • O login pode ser feito por e-mail/senha ou conta Google.
  • Dados de assinatura são usados para liberar recursos Premium; dados de cartão e Pix devem ser processados pelo gateway de pagamento.
  • O site pode utilizar cookies e tecnologias semelhantes para funcionamento, segurança, análise e publicidade.
  • O FinanPro não vende dados pessoais dos usuários.

Ao utilizar o FinanPro, você concorda com esta Política de Privacidade.
